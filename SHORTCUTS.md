# Terminal

| Shortcut | Action |
| --- | --- |
| `<Ctrl>s`						| Scroll lock (XOFF) |
| `<Ctrl>q`						| Scroll unlock (XON) |
| `<Ctrl><Mouse drag>`			| Block selection |

# Caja

| Shortcut | Action |
| --- | --- |
| `F3`							| Show/hide extra pane |
| `<Ctrl>l`						| Show editable location text field |

| `F7`							| Toggle sidebar |
